Программа для выявления присутствия человека в опасной зоне на предприятии по кадрам с камер наблюдения.
Для выполнения программы необходимо открыть файл app_2.ipynb 
В низу кода указать путь к папке с датасетом в переменную dataset_folder
Также указать путь в переменную output_csv_path для сохранения файла CSV , в котором будут храниться результаты работы модели, для указанного датасета.
Запустить программу 
Результат работы будет сохранен по указанному пути в переменной output_csv_path
